# DINAMIS Web-App auf GitHub Pages veroeffentlichen

## Was du brauchst
- Einen GitHub-Account (kostenlos: github.com/signup)
- Deinen Browser (am Mac/Laptop)

## Schritt 1: GitHub Repository erstellen

1. Gehe zu **github.com** und melde dich an
2. Klicke oben rechts auf **+** und waehle **New repository**
3. Name: `dinamis` (oder ein anderer Name deiner Wahl)
4. Visibility: **Public** (muss Public sein fuer kostenloses GitHub Pages)
5. Klicke **Create repository**

## Schritt 2: Dateien hochladen

1. Im neuen Repository klicke auf **uploading an existing file**
2. Ziehe alle 3 Dateien aus dem `dinamis-web` Ordner hinein:
   - `index.html`
   - `eingabe.html`
   - `ausgabe.html`
3. Klicke unten auf **Commit changes**

## Schritt 3: GitHub Pages aktivieren

1. Gehe zu **Settings** (Zahnrad-Symbol im Repository)
2. Klicke links auf **Pages**
3. Unter "Source" waehle **Deploy from a branch**
4. Unter "Branch" waehle **main** und Ordner **/ (root)**
5. Klicke **Save**

## Schritt 4: Deine URL

Nach 1-2 Minuten ist deine App online unter:

**https://DEIN-USERNAME.github.io/dinamis/**

Ersetze `DEIN-USERNAME` durch deinen GitHub-Benutzernamen.

Diese URL funktioniert auf jedem Geraet: iPhone, iPad, Mac, Windows, Android.

## Hinweise

- Alle Daten bleiben lokal im Browser. Es wird nichts an einen Server gesendet.
- Die App funktioniert komplett offline, sobald sie einmal geladen wurde.
- Um Aenderungen zu machen, lade einfach die neuen HTML-Dateien im Repository hoch.
